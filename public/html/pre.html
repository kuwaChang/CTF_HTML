：） CTFとは？

CTF（Capture The Flag）は、サイバーセキュリティのスキルを競う競技です。

各問題には「フラグ」と呼ばれる答えが隠されており、暗号解読、ファイル解析、Webアプリケーションの脆弱性発見など、様々な技術を使ってフラグを見つけ出します。

このサイトでは、ストーリー仕立ての問題を通じて、CTFの基礎から応用まで楽しく学ぶことができます！

：） このサイトでの学習方法

このサイトには、Easy1、Easy2、Medium1、Medium2、Hard1、Hard2という6つの独立したストーリーが用意されています。

✨ 重要：各ストーリーは完全に独立しています

どのストーリーから始めても構いません。好きなストーリーを選んで、楽しくCTFを学びましょう！

📚 各ストーリーについて

それぞれのストーリーには、独自の世界観と物語が用意されています：

  • Easy1：AI「ノア」が閉じ込められているシステムから脱出する物語
  • Easy2：探偵事務所で様々な依頼を解決していく物語
  • Medium1：失われた文明「アトラス」の遺跡を探索する物語
  • その他、それぞれ異なる世界観とストーリーが展開されます

各ストーリー内では、複数の問題（通常5問程度）を解いていくことで、ストーリーが進んでいきます。

問題は難易度順に並んでいるわけではないので、「これなら解けそう」と思う問題から挑戦してください！

🎯 初心者の方へ

最初から難しい問題に挑戦すると、解き方や問題に対する取り組み方がわからないと思います。

そのため、Easy1またはEasy2から始めることをおすすめします。

準備ができたら、どれかを選んでCTFに挑戦してみましょう！

：） 環境構築

WSLをインストールしよう

Windows PowerShellを管理者権限で起動して、以下のコマンドを実行しよう：

wsl --install

※WSL（Windows Subsystem for Linux）は、Windows上でLinux環境を使えるようにする機能です。CTFの問題を解く際に便利なツールが使えるようになります。

：） CTFカテゴリーの簡単な説明

CTFの問題は、主に以下のカテゴリーに分類されます。各ストーリーでは、これらのカテゴリーの問題が登場します。

Crypto -暗号-

概要 暗号アルゴリズムや実装の弱点を突いて平文や鍵を取り出す問題
典型問題 シーザー・Vigenère・RSAの誤実装、OTPのミス、ハッシュ衝突、エンコードの積み重ね
最初に試すこと 与えられた文字列をエンコード・デコード（base64, hex, rot13）で確認し、よくある鍵長やパターンを探す

Forensics

概要 ファイルやメモリ、ネットワークキャプチャから痕跡を解析して証拠（フラグ）を見つける分野
典型問題 PCAP解析、画像・PDFのステガノ、タイムスタンプ解析、ディスクイメージの復元、メモリダンプ解析
最初に試すこと ファイルタイプ確認（file）、ヘッダを見る、metadataを調べる（Exif）、stringsでテキスト抽出

Reversing

概要 バイナリ（実行ファイル）の動作を逆解析して、挙動や隠しロジック・キーを見つける
典型問題 キー検証ルーチンの解析、難読化解除、パッチでフラグを出す、オフラインライセンス解除
最初に試すこと 実行して挙動確認（入力・出力）、strings → IDA/Ghidraで関数を見る、難易度によって動的解析（デバッガ）へ

Web

概要 ウェブアプリの脆弱性（認証、入力検証、セッション管理など）を突く問題
典型問題 SQLインジェクション、XSS、CSRF、ファイルアップロードの不備、IDOR（不適切な直接参照）
最初に試すこと 機能を洗い出す（ログイン、ファイル、アップロード、API）、入力に対してどこまでサニタイズされているか試す

Pwn

概要 バイナリの脆弱性（バッファオーバーフロー、フォーマットストリング等）を利用して任意コードを実行する分野
典型問題 バッファオーバーフローでリターンアドレスを書き換える、ROPチェーン、ASLR/DEP回避、フォーマット文字列でメモリ書換
最初に試すこと ローカルで実行して脆弱性を再現、gdbでクラッシュ箇所を特定、簡単なペイロードで挙動確認

OSINT

概要 公開されている情報（SNS、WHOIS、Webアーカイブ等）を組み合わせて人物や組織の情報を見つける技術
典型問題 ある人物のメールアドレスやサブドメインを探す、過去のページから手がかりを得る、画像のメタデータ解析で場所特定
最初に試すこと 名前・ハンドル・メールの断片で検索、Google dork、Wayback Machineで過去のサイトを調査
