# OSINT

## OSINTとは何か
オープン・ソース・インテリジェンス（英: Open-Source Intelligence）とは、  
**合法的に入手可能な公開情報を収集・分析し、価値ある情報を導き出す手法**である。  

略称は **OSINT（オシント）**。  
また、**Open-Source Investigation（公開情報調査）**とも呼ばれる。

## CTFでよく問われる具体的な対象

### 画像解析（フォレンジック系の基本）

#### 何を見る？
- 画像の中に隠されたデータ
- ファイル構造（ヘッダ・フッタ・チャンク）
- 見た目では分からない埋め込み情報

#### よくある問題
- 画像内にフラグが埋め込まれている
- LSB（最下位ビット）によるステガノグラフィ
- 画像の末尾にZIPやテキストが付加されている

#### よく使うツール
- `strings`：文字列抽出
- `binwalk`：埋め込みファイル検出
- `stegsolve`：可視化・ビット解析
- `zsteg`：PNG/BMPのステガノ検出

### 位置特定（OSINTの定番）

#### 何を見る？
- 建物・看板・言語
- 地形（山、海、川）
- 車のナンバー・道路標識
- 電柱・信号機・道路の特徴

#### よくある問題
- 「この写真はどこで撮られたか？」
- 「この建物の名前は？」
- 「この場所の座標を答えよ」

#### よく使うツール
- Google Maps / Street View
- Google Lens / 画像検索
- OpenStreetMap

### メタデータ（EXIFなど）

#### 何を見る？
- 撮影日時
- GPS座標（緯度・経度）
- 使用カメラ・デバイス情報

#### よくある問題
- EXIFのGPSから撮影場所を特定
- 撮影時間から行動を推測
- 編集履歴から改ざんを疑う

#### ツール
- `exiftool`

### ファイル解析（中身を見る）

#### 何を見る？
- 実際のファイル形式（拡張子との不一致）
- ファイル内部構造（ヘッダ・マジックナンバー）
- 埋め込みデータや分割データ

#### よくある問題
- `.jpg`に見せかけた`.zip`
- 1つのファイルに複数のファイルが埋め込まれている
- 壊れたファイルの修復

#### ツール
- `file`：ファイル形式判定
- `binwalk`：埋め込み抽出
- `xxd` / `hexdump`：バイナリ解析
- `foremost` / `scalpel`：ファイルカービング

### 文字列・エンコード

#### 何を見る？
- 意味のありそうな文字列
- 規則性のある英数字列
- フラグ形式（例：`FLAG{...}`）

#### よくある問題
- Base64 / Base32 / Base58
- ROT13 / Caesar暗号
- Hex / URLエンコード

#### ツール
- `strings`
- CyberChef
- `base64`
- `xxd`

### Web情報（OSINT）

#### 何を見る？
- HTMLソースコード
- コメントアウトされた情報
- `robots.txt`
- JavaScriptファイル
- 通信内容（Networkタブ）

#### よくある問題
- ソースコードにフラグが隠れている
- `robots.txt`から隠しディレクトリを発見
- APIレスポンスにヒントがある

#### ツール
- ブラウザ開発者ツール（F12）
- `curl` / `wget`
- Burp Suite
- `dirsearch` / `gobuster`

## 重要な考え方（初心者向け）

### 1. まず全部出す
- strings
- exif
- file

### 2. 違和感を探す
- 不自然なデータ
- 変な文字列
- 拡張子と中身のズレ

### 3. 仮説 → 検証
- 「Base64っぽい」→ デコード
- 「zipっぽい」→ 解凍

## 最低限の初動テンプレ

1. `file` → ファイルの正体確認  
2. `strings` → 文字列抽出  
3. `exiftool` → メタデータ確認  
4. `binwalk` → 埋め込み探索 

## OSINT / フォレンジック手順まとめ

### 1. 画像の逆引き検索

#### 使用ツール
- Google Images（画像検索）
- TinEye
- Yandex Images（建物・人物に強い）

#### 手順
1. 画像をアップロードして検索
2. 完全一致と類似画像を確認
3. 最も古い投稿（初出）を探す
4. 別角度の画像を収集する
5. 写っている要素（建物・看板など）を特定する

#### ポイント
- GoogleだけでなくYandexも使う
- 「最初にどこで出たか」を重視する

### 2. 位置特定（ジオロケーション）

#### 使用ツール
- Google Maps
- Google Street View
- GeoGuessr（練習用）

#### 手順
1. 画像からヒント抽出（言語・ナンバープレート・電柱・信号など）
2. 地域を仮決めする
3. Mapsで候補を探す
4. Street Viewで一致確認する
5. 建物配置・道路・影を照合する

#### ポイント
- 影の向きから方角を推定できる
- 看板の電話番号から地域を絞れることがある

### 3. メタデータ解析（EXIF）

#### 使用ツール
- ExifTool

#### 基本コマンド
```bash
exiftool image.jpg
```

#### 確認項目
- GPS情報
- 撮影日時
- カメラ機種
- 編集履歴

#### 応用コマンド
```bash
exiftool -gps* image.png
```

#### ポイント
- SNSではEXIF情報が削除されることが多い
- 加工履歴が残る場合がある

### 4. ドメイン調査（WHOIS/DNS）

#### 使用ツール
- `whois`
- `nslookup`
- `dig`

#### 手順
```bash
whois example.com
```

#### 確認項目
- 登録者情報
- 作成日
- ネームサーバ

#### 追加調査
```bash
nslookup example.com
dig example.com
```

#### ポイント
- ドメインの新旧は信頼性判断の材料になる
- 同一IP上の他サイトも確認する

### 5. ユーザ名調査

#### 使用ツール
- Sherlock
- Namechk

#### 手順例
```bash
python sherlock username
```

#### 調査手順
1. 同一IDのSNSを列挙する
2. 投稿内容・時間帯を確認する
3. 共通情報（地域・趣味など）を抽出する

#### ポイント
- IDの使い回しは多い
- GitHub / Redditなども有効な情報源になる

### 6. ステガノグラフィ調査

#### 使用ツール
- `binwalk`
- `strings`
- `zsteg`

#### 手順
```bash
binwalk image.png
strings image.png
zsteg image.png
```

#### ポイント
- 末尾にZIPが付いていることが多い
- PNGはLSBを疑う

### 7. ファイルフォレンジック

#### 使用ツール
- `file`
- `xxd`
- `hexdump`

#### 手順
```bash
file suspicious.bin
xxd suspicious.bin | head
```

#### ポイント
- 拡張子偽装に注意する
- マジックナンバー確認が重要

### 8. OCR（文字認識）
- 画像中の文字をテキスト化する技術

#### 代表ツール
- Tesseract OCR

```bash
tesseract image.png output
```

| OCR | ステガノ |
| --- | --- |
| 見えている文字を読む | 見えないデータを探す |

## OSINTの適切な利用

### 原則
- 公開情報のみを使う
- 目的を明確にする
- 必要最低限の収集にとどめる
- 結果の扱いに責任を持つ

### プライバシー配慮の基本
1. 個人特定を安易にしない（本名・住所・勤務先を不要に結び付けない）
2. 二次利用に注意する（収集情報を再配布しない）
3. 文脈を歪めない（切り取りや古い情報の誤用を避ける）
4. 本人の意図を考える（公開情報でも拡散前提とは限らない）

### 守るべきもの
- 利用規約
- 著作権
- 個人情報保護
- 不正アクセス禁止法

## EXIF情報とは
EXIF（Exchangeable Image File Format）とは、画像ファイルに埋め込まれているメタデータ（付加情報）のこと。

### 代表的な情報
- GPS位置情報（緯度・経度）
- 撮影日時
- カメラ機種（iPhone・Canonなど）
- 撮影設定（ISO・シャッター速度など）
- 編集ソフト（Photoshopなど）

### 確認方法
代表ツールは`ExifTool`。

```bash
exiftool image.png
```

### 注意点
1. EXIFはSNS等で削除されることが多い
2. EXIFは書き換え可能
3. 位置情報漏えいなどのプライバシーリスクがある
